# 📞 Customer Call Center AI Project

## 📌 Overview
This project is an AI-based system that processes and classifies customer call data using Natural Language Processing (NLP) and Machine Learning. It helps automate customer query handling in call centers.

## 🎯 Objective
- Analyze customer call data
- Classify customer queries into categories
- Improve response efficiency using AI

## 🧠 Technologies Used
- Python
- Jupyter Notebook
- NLP (Text Processing)
- Machine Learning (Naive Bayes)

## ⚙️ Features
- Data preprocessing
- Text feature extraction
- Model training and prediction
- Customer query classification

## 📂 Project Structure
customer-call-center-project/
│── SmartNode_POC.ipynb  
│── calls_audio/ (not uploaded due to size limitations)

## 📊 Dataset
Audio call data was used for training and testing.  
(Note: Dataset is not uploaded due to large file size.)

## ▶️ How to Run
1. Install dependencies:
pip install -r requirements.txt

2. Open Jupyter Notebook:
jupyter notebook

3. Run the file:
SmartNode_POC.ipynb
